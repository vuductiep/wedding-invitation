# Wedding Music

To add music to your wedding invitation:

1. Replace this README.md file with your audio file
2. Name the audio file: `wedding-music.mp3` (or update the src attribute in the audio element in wedding-invite.tsx)
3. Supported formats: MP3, WAV, OGG
4. The audio will loop automatically and play at 50% volume

## Recommended Specifications:
- Format: MP3 (for best browser compatibility)
- Duration: Loop-friendly track (30 seconds to several minutes)
- Volume: Already set to 50% in the code, adjust in wedding-invite.tsx if needed

## Example:
If you have a song called "wedding-song.mp3", rename it to "wedding-music.mp3" and place it in this directory.
